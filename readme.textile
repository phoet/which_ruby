h2. About

*which_ruby* is a simple helper for checking running Ruby engine.

h2. Usage

include WhichRuby:
<pre>
rvm use jruby
gem install which_ruby
irb
 > require 'which_ruby'
=> true 
 > include WhichRuby
=> Object
</pre>

access the API:
<pre>
 > ruby_type
=> :jruby
 > rt
=> :jruby
 > is_ruby_type? :jruby
=> true 
 > r? :jruby
=> true 
 > jruby?
=> true
</pre>

h2. Examples

Run the examples with different runtimes, jruby for example:

<pre>
rvm use jruby
ruby examples/ex.rb
</pre>
