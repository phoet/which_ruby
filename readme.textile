h2. About

*which_ruby* is a simple helper for checking running Ruby engine

h2. Usage

<pre>
rvm use 1.8.7
gem install which_ruby
irb
 > require 'which_ruby'
=> true 
 > include WhichRuby
=> Object 
 > ruby_type
=> :ruby 
 > is_ruby_type? :ruby
=> true 
 > r? :ruby and rt == :ruby
=> true
</pre>
